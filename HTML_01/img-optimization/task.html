Instructions & Hints

  • Assume the gallery layout: on small screens, each image is 100vw. On large screens, each image is about 30vw.
  • Use the sizes attribute to tell the browser about this layout. e.g., sizes="(max-width: 600px) 100vw, 30vw"
  • For the bonus task: Remember the structure of the <picture> element. Offer WebP first, then JPG as a fallback.
  • Check your work in the browser's DevTools Network tab! Simulate a mobile device and see which image file loads.
