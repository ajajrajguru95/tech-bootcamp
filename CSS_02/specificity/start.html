The Specificity Problem

Which styles will actually apply? The order seems random!

Specificity Hierarchy (Most to Least Powerful):

  1. !important (avoid when possible)
  2. Inline styles (style attribute)
  3. ID selectors (#submit-btn)
  4. Class selectors (.primary-button)
  5. Element selectors (button)

Specificity Calculator

Try these selectors:

  • div = (0,0,1)
  • .container = (0,1,0)
  • #main = (1,0,0)
  • div#main.container = (1,1,1)

Specificity Score Calculation:

button = (0,0,1) - 1 point for element

.primary-button = (0,1,0) - 10 points for class

#submit-btn = (1,0,0) - 100 points for ID

Higher score wins!
