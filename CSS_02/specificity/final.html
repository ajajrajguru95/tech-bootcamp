CSS Specificity - Controlled!

Specificity Score Calculation:

button = (0,0,1) - 1 point for element

.primary-button = (0,1,0) - 10 points for class

#submit-btn = (1,0,0) - 100 points for ID

Higher score wins!

Best Practices:

  • Use classes for most styling
  • Avoid using IDs for CSS when possible
  • Never use !important unless absolutely necessary
  • Keep specificity low and consistent
