Where Margins Fit in the Box Model

Box Without Margin
Total width: 200px (border-box)
Margin: 0px
Box With Margin
Total width: 200px (border-box)
Margin: 30px each side
Space taken: 200 + 30 + 30 = 260px

The Complete Formula:

With border-box:
Total Space = width + margin-left + margin-right

With content-box:
Total Space = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
